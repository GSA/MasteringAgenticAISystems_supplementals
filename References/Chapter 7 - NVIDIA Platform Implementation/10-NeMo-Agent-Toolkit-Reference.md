## Toolkit Overview

NeMo Agent Toolkit provides:

**Core Components:**
- Framework-agnostic agent development
- Pre-built agents for common tasks
- Automatic hyperparameter tuning
- Google ADK (All-Devices Kit) support

**Key Features:**
- Flexible architecture for custom agents
- Integration with multiple LLM providers
- Model Context Protocol (MCP) support
- Authorization and security framework
